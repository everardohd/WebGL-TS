# WebGL-TS
WebGL Typescript project for learning purposes

original project:
    https://developer.mozilla.org/en-US/docs/Web/API/WebGL_API/Tutorial/Animating_textures_in_WebGL

Dependencies:
* NPM package manager
> https://nodejs.org/es/
* Typescript
> http://www.typescriptlang.org

Run:
> npm install

> tsc

> npm run dev
 
Tested on:
* Chrome
* Edge
* Firefox
